# manhuadb 漫画下载

- the url format: https://www.manhuadb.com/manhua/xxx

## Usage

### Prepare

```shell
pip install -r requirements.txt
```

### Download

```shell
python manhuadb.py [url] [path] 
```

the [path] arg is optional
